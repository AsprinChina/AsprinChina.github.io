无约束问题的优化，就是给定一个函数求取其最大/小值。按照优化算法是否利用了这个函数的导数（梯度、海森矩阵），相应算法可以分为利用导数的优化算法（下降法、牛顿法、……）和不利用导数的优化算法（模式搜索法、Powell法、……）。鉴于实际应用中前者效果更好，这里介绍其中最为常用的梯度下降法、牛顿法、拟牛顿法、共轭梯度法。

1. 无约束优化的最优性条件

对于一个优化问题，若要找到其最优解，首先就要明确最优解有什么性质，满足什么条件，然后才能够据此去推导求解的算法。最优解满足的条件（必要、充分、充要）称为最优性条件。（戳这里回顾）

通常，我们把这一个必要条件作为收敛判据，即$|\nabla f(\mathbf {x^k})|_2\le\epsilon$时结束迭代。对于凸优化问题，此时找到的解$\mathbf {x^k}$就是全局最优解（严格来说是无限接近最优解的次优解，详见Boyd在[1]中给出的证明）；而对于非凸优化问题，这个必要条件只能保证$\mathbf {x^k}$是驻点（极大点、极小点、鞍点），通常极大点与鞍点都可以被排除，而无法区分极小点$\mathbf {x^k}$是全局最优解还是局部最优解。这也正是非凸问题远远难于凸问题的原因。

2.1. 梯度下降法

2.1.1. 原理推导

梯度下降法是最常用的无约束优化算法，具有一阶收敛性。特点是迭代格式简单，但是收敛速度较慢。

2.1.3. 变体

梯度下降法在深度学习领域有很多变体，包括小批量梯度下降（Mini-Batch Gradient Descent）、随机梯度下降（Stochastic Gradient Descent）、冲量梯度下降（Momentum Gradient Descent）、Nesterov加速梯度法（NAG）、自适应矩估计方法（Adam）等。

因为具体内容过多，所以后续单独开一个Blog来讨论这些算法，这里简要介绍MBGD、SGD、MGD的思想，不涉及相关公式。

上述变体出现的主要背景是深度学习中学习样本的海量增加（所谓大数据）。我们把每个样本记为$(\mathbf X, y)$，学习任务就是要拟合出一个函数$f(\mathbf X)$，使得某个关于$f(\mathbf X)$和$y$的损失函数，如均方误差损失函数$\frac 1N\sum_{i=1}^{N}(f(\mathbf X)-y)^2$，最小化。所以这是一个无约束优化问题，在$N$不大的时候梯度下降方法并没有暴露出太大的问题。而当样本数$N$很大时，每一步迭代都要把所有样本算一次，导致迭代速度很慢。

（1）小批量梯度下降

顾名思义，小排量梯度下降方法就是把所有样本（比如说10000个）分成若干批（比如说100批，每批100个），然后每步梯度下降迭代都只用其中1批样本计算梯度，那么迭代速度就可以大大加快。这一变体并未涉及到梯度下降迭代格式的改变，仅仅只是其中的源数据使用方式发生了改变。

（2）随机梯度下降

随机梯度下降更为极端，在每一步迭代中只采用1个样本进行梯度计算，每轮迭代时间极大缩短，但是极易发生震荡问题。显然，这一方法也没有改变梯度下降的迭代格式，也是仅改变源数据的使用方式。

（3）冲量梯度下降

当优化问题的数值条件不太好时，比如说各维度数值尺度差异很大，如图，那么梯度下降在迭代的时候就会发生震荡前进的现象，导致实际迭代速度并不快。冲量梯度下降的基本思想就是说，在确定每一步的前进方向时，并不直接用当前点处的负梯度方向作为前进方向，而是用上一步的前进方向和当前负梯度方向的加权平均来作为当前的前进方向（有点滤波的感觉），这样那些随机震荡的方向（比如图中的上下）就会相互抵消，而真正的前进方向由于每步负梯度方向都有而被加强，从而加快收敛。

扯一点题外话，尽管那么多做优化的前人提了这么多超线性的收敛方法（后面会介绍其中3种），但是在深度学习领域，优化问题（最小化损失函数）的求解还是清一色的梯度下降（加上各种定语）……超线性的方法反而存在这样那样的问题。曾经有一个前辈在讲座中评论这个，大意是说，作为一个做优化的，他觉得，不是这些超线性方法不适合这些问题，而是还没有找到运用这些方法去解决这些问题的合适途径。

2.2. 牛顿法

2.2.1. 原理推导

牛顿法是一种求解代数方程组$F(\mathbf x)=\mathbf 0$的数值算法，因为优化问题可以通过最优性条件转化为一组代数方程（零梯度方程），所以牛顿法也可以用于求解优化问题（二阶优化算法，速度快）。

再来说说为什么牛顿法收敛快的直观理解。

梯度下降法用最速下降方向作为当前点处的前进方向，实质上是一种贪心算法；而牛顿法利用二阶导数信息看得更远，不仅要当前下降快，还要走了一步之后下降也快，这是通过新增的海森矩阵逆矩阵对原来的最速下降方向进行修正实现的。牛顿法与梯度下降法寻优的示意图如图。

首先，对比梯度下降法的求解轨迹与牛顿法的求解轨迹，直观发现牛顿法的迭代点数量远远少于梯度下降法，实例证明了牛顿法迭代速度远快于梯度下降法。

而对于问题2（非凸），牛顿法也表现为初值敏感。图中给定的三个初值点，分别收敛到了极小点、极大点和鞍点。需要说明的是，梯度下降法是不会收敛到极大点的，而牛顿法可能收敛到极大点。根本原因是牛顿法的推导完全是由零梯度代数方程推得的，所以算法只能保证最后的收敛解处梯度为零。直接原因是牛顿法对最速下降方向进行了修改，使得牛顿法的前进方向可能不是下降方向（即不满足$\nabla f(\mathbf x)^T\Delta\mathbf x<0$）。

除此之外，牛顿法的收敛性也不太好，在很多初值点处均出现了迭代发散的情况。

上述问题均表明：牛顿法需要改良。于是，就出现了一系列牛顿法的变体。

2.2.3. 变体

（1）阻尼牛顿法

牛顿法的步长固定为1，这在一定程度上加剧了算法的发散问题。针对这个问题，阻尼牛顿法在牛顿法基础上，增加一个可变步长$\lambda_k$，并在每轮迭代中通过一维搜索来确定相应取值。

修改后的阻尼牛顿法算法流程：

输入：梯度$\phi(x)$，海森矩阵$H(x)$，收敛阈值$\epsilon$，初始点$x_0$
输出：局部极小点$\hat x$
1. 输入，并置初始梯度二范数$\begin{Vmatrix}\phi(x) \end{Vmatrix}_2=2\epsilon$
2. while($\begin{Vmatrix}\phi(x) \end{Vmatrix}_2>\epsilon$):
3. $\ \ \ \ $ $\lambda_k=arg\,\min_{\lambda}\ {f(x-\lambda H(x)^{-1}\phi(x))}$
4. $\ \ \ \ $ $x=x-\lambda_kH(x)^{-1} \phi(x)$
5. 输出$x$

（2）修正牛顿法

牛顿法的迭代格式（$x=x-H^{-1} \phi(x)$）中涉及到海森矩阵求逆。此时，如果海森矩阵奇异，那么算法就无法继续。

另外，牛顿法的前进方向还可能不是下降方向，导致最终可能收敛至极大点。这一问题已经暴露在上面问题2的求解中。现在给出详细的论述：

若牛顿方向【$-H^{-1} \nabla f(x)$】，对任意梯度$\nabla f(x)$满足

即海森矩阵$H$正定时，则牛顿法能够保证每一步的前进方向都是下降方向。然而，牛顿法中的海森矩阵$H$往往无法满足这一要求，导致前进方向可能变成上升方向。

（3）拟牛顿法

注意到，牛顿法在每次迭代的时候都要更新海森矩阵，并求逆，这个工作的计算量是巨大的。

针对这个问题，拟牛顿法就是试图通过某种方法，利用梯度信息来直接构造海森矩阵的逆矩阵，从而减小每轮迭代的计算量。这一构造方法有很多，由此也就形成了不同的拟牛顿法。因为内容较多，所以单独开一节介绍拟牛顿法这一类方法。

2.3. 拟牛顿法

上面说到牛顿法有两大问题

  • 每步迭代更新海森矩阵并求逆矩阵，计算量极大
  • 海森矩阵可能非正定，导致牛顿方向不保证是下降方向，甚至可能出现收敛至极大点。

2.3.1. 秩1校正法

无论秩1校正法，还是DFP或BFGS，其对$H$的构造方式都是给定一个初始矩阵$H_0$，然后在第$k+1$轮迭代中对上一轮的近似矩阵$H_k$作出修正（$+\Delta H_k$）从而得到满足本轮迭代的拟牛顿条件的近似矩阵$H_{k+1}$。

对于秩1校正法，第$k+1$轮迭代的校正矩阵定义为

将$H_{k+1}=H_k+\Delta H_k$代入拟牛顿条件可以证明，对$\forall$给定的$H_0$，每轮迭代中用上式修正的$H_{k+1}$均满足拟牛顿条件。

之所以该方法称为秩1校正，是因为该校正矩阵$\Delta H_k$是一个秩为1的矩阵（这一结论由其推导过程可以直观得出，见 [2]）。

需要注意的是，秩1校正法并不保证$H_k$的正定性。仅当$q_k^T(p_k-H_kq_k)>0$成立时，$H_k$保证正定性。

对于DFP算法，第$k+1$轮迭代的校正矩阵为

将$H_{k+1}=H_k+\Delta H_k$代入拟牛顿条件可以证明，对$\forall$给定的$H_0$，每轮迭代中用上式修正的$H_{k+1}$均满足拟牛顿条件。

该方法之所以称为DFP算法，是因为该方法由Davidon提出，再由Fletcher和Powell改进。

DFP算法构造的近似矩阵$H$能够保证其正定性，同时对于二次规划问题有二次终止性（即有限步收敛到最优解）。

2.3.3. BFGS公式

对于BFGS算法，第$k+1$轮迭代的校正矩阵为

将$H_{k+1}=H_k+\Delta H_k$代入拟牛顿条件可以证明，对$\forall$给定的$H_0$，每轮迭代中用上式修正的$H_{k+1}$均满足拟牛顿条件。

该算法之所以称为BFGS算法，和DFP是一个原因，是时候祭出这张广为流传的照片了。

BFGS的推导也很有意思，把DFP公式中的$p_k$和$q_k$互换，然后对那个$H_{k+1}$求逆，就是BFGS公式。所以BFGS公式也称DFP公式的对偶公式，详细推导见 [2]。

和DFP一样，BFGS公式也保证构造的$H$的正定性。同时，数值计算经验表明BFGS公式比DFP公式更好。

拟牛顿法行之有效，但也面临内存需求大等困难。工程上常用Limited Memory - BFGS（LBFGS）来实现BFGS算法，区别在于对中间变量的存储管理（不放到内存中，即用即算，以时间换空间），而不涉及算法原理的改变。

2.4. 共轭梯度法

在介绍共轭梯度法之前，首先要介绍一下什么是共轭方向。

对于$n$阶对称正定矩阵$A$，若$\mathbb R^n$中的两个方向列向量$d_1$、$d_2$满足

$A$的所有共轭方向$d_1$、$d_2$、……、$d_k$（两两关于$A$共轭）是线性无关的，且有扩张子空间定理成立：

对于函数$f(\mathbf x)=\frac 12 \mathbf {x^T} A \mathbf x + \mathbf {b^T} \mathbf x +c$，以$\forall \mathbf {x_0} \in \mathbb R^n$为初始点，依次沿$d_1$、$d_2$、……、$d_k$进行一维搜索，最终得到$\mathbf {x_k}$是$f(\mathbf x)$在线性流形$\begin{Bmatrix} \mathbf x|\mathbf x=\mathbf {x_0} + \sum_{i=1}^k\lambda_id_i,\ \lambda_i\in R \end{Bmatrix}$上的唯一极小点。特别地，当$k=n$时，$\mathbf {x_n}$是$\mathbb R^n$上的唯一极小点。

2.4.1. 原理推导

扩张子空间定理保证了，对于二次凸函数，经一组共轭方向进行一维搜索，可在有限步内找到最小点。共轭梯度法正是基于这一思想，对梯度下降法的最速下降方向进行改造，修正为一组共轭方向，从而保证二次终止性。

实际上，共轭梯度法最开始提出（1952年）是用于求解线性方程组的，后来经改造用于无约束优化问题的求解。同拟牛顿法一样，共轭梯度法也是一类算法，根据修正格式的不同有FR法、PRP法等（这些字母也都是提出人的姓名首字母）。

这里推导FR（Fletcher-Reeves）共轭梯度法的原理，首先基于二次凸函数$f(\mathbf x)=\frac 12 \mathbf {x^T} A \mathbf x + \mathbf {b^T} \mathbf x +c$展开，然后再推广到一般函数的无约束优化上去。

对于该凸二次函数，记第$k$轮迭代从点$x_k$出发，搜索方向为$d_k$，前进步长为$\lambda_k$，到达点$x_{k+1}$。在点$x_{k+1}$处，要用$d_k$和当前负梯度方向$-\nabla f(\mathbf x_{k+1})$构造本轮迭代的前进方向$d_{k+1}$以保证$d_k$和$d_{k+1}$关于$A$共轭。

有两点需要注意：

第一点需要注意的是，在上述推导过程中，我们只保证了相邻两次迭代的前进方向关于$A$共轭，若要求所有迭代方向两两共轭，则初始迭代方向必须是负梯度方向。

第二点需要注意的是，共轭梯度法在推广到一般函数的优化上时，就没有了有限步终止的性质。此时，搜索完所有$n$个共轭方向之后，有两种迭代继续的方案：

  • 继续按照共轭梯度法的迭代格式继续迭代；
  • 迭代完$n$个共轭方向之后，让前进方向回到负梯度方向，重新遍历$n$个共轭方向。

计算经验来看，后者更好些。

2.4.2. 算法实现

虽然FR共轭梯度法的推导中要求了一维搜索，但是在推广到一般函数优化之后，已经没有了有限步终止性质，所以也没有必要一定要求通过一维搜索来确定步长。出于便利，这里将搜索步长固定为常数。要注意的是，当不再采用一维搜索时，共轭梯度法给出的前进方向不再保证是下降方向。

具体地，（固定步长的）共轭梯度法的算法流程为

输出：局部极小点$\hat x$
1. 输入，并置初始梯度二范数$\begin{Vmatrix}\phi(x) \end{Vmatrix}_2=2\epsilon$
2. 置$d=-\phi(x_0)$
3. while($\begin{Vmatrix}\phi(x) \end{Vmatrix}_2>\epsilon$):
4. $\ \ \ \ $如果迭代次数不是变量维数$n$的整数倍：
5. $\ \ \ \ \ \ \ \ $计算修正系数$\beta$（FR\PRP\Daniel……）
6. $\ \ \ \ \ \ \ \ $修正前进方向为共轭方向$d=-\phi(x)+\beta d$
7. $\ \ \ \ $否则：
8. $\ \ \ \ \ \ \ \ $设置前进方向为负梯度方向$d=-\phi(x)$
9. $\ \ \ \ $$x=x+\lambda d$
10. 输出$x$
